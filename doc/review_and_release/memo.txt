October 5, 2007

TO: IPAC/OCEC

FROM: Christopher J. Morrone, Livermore Computing

SUBJECT: Memorandum For Record for Software Release

Livermore computing is requesting the release of a software tool
called "libyogrt" which is used for determining a parallel application's
remaining time in its job allocation.  We are requesting the release of
this software because it is useful to others in HPC community.  Releasing
this software will allow code groups across the tri-labs to standardize
on one interface.  We wish to release this software under the GNU Lesser
General Public License.
